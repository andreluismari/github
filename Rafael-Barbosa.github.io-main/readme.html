Como deletar uma branch?
Da pra deletar localmente com o comando git branch -d nome-da-branch
Entre na main branch e faça um merge da branch desejada para integrar ao main. Para resolver o conflito do código, por que é preferível utilizar o rebase em vez do merge?
O rebase é preferível ao merge em algumas situações porque:
1 - o rebase mantém um histórico linear mais limpo, o que facilita o entendimento das mudanças e revisões
2 - evita a criação de commits de merge, o que pode complicar o histórico de commits
3 - permite reaplicar mudanças de uma branch em cima de outra, como se tivesse sido feita diretamente no ponto mais recente da branch base.
Porém o rebase deve ser usado com cuidado, especialmente em branches compartilhadas, para evitar complicações com o histórico compartilhado.

Como inserir apenas um commit de uma branch para a main?
Você pode utilizar o cherry-pick, vendo no log do git vai ter um hash dos commits, pegue o log dele e de checkout na branch qual você quer realizar o commit, e rode o comando git cherry-pick #hash
